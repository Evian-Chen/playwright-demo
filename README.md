# playwright-demo
This repository is used to test `Playwright`.

## run file
If you want to run the smallest test case without rendering the UI, use
```
npm test
```

If you want to see open brower, run
```
npm run test:headed
```

## folder architecture
The basic folder architecture as follow:
```
playwright-demo/
├─ package.json
├─ tsconfig.json
├─ tests/
│  └─ example.spec.ts
└─ node_modules/
```
